# Обучающий проект для изучения Python

## Описание:

Проект представляет из себя виджет для работы над банковскими операциями клиентов.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Rayour/BankProject.git
```
2. Установите зависимости:
```
poetry instal 
```
